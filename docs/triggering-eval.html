How the router is scored

The eval reads each skill's live description from its SKILL.md, so the test can't drift from the skills.

1 · skill descriptions Read straight from every SKILL.md — the same text Claude Code triggers on.
▶
2 · router model Given a prompt, picks the single best-matching skill — or none.
▶
3 · score vs. fixture Pick compared to the expected label; prints per-case ✓/✗ and an accuracy total.
